# 待测试

本仓库自动化已经覆盖前后端构建、单元测试、真实 PostgreSQL/Redis 集成、Session/RBAC、100 账号导入、额度账本、Provider 密钥脱敏、备份恢复、40 人并发和跨设计师素材隔离。

正式公司环境仍需完成：

- 企业微信真实 Corp ID、Agent ID、Secret 和 HTTPS 回调域验收。
- 公司正式域名、WAF/反向代理、证书续期和可信代理配置。
- 公司私有云或 S3 兼容对象存储的容量、生命周期、备份和恢复演练。
- 每个真实 Provider 的限流、超时、失败退款和成本核对。
- RunningHub、ComfyUI 与公司内部工作流的真实请求/状态/结果路径验收。
- 10 人试点后逐步扩大到约 100 位设计师，并复测部门权限和审计导出。
- 移动端画布触控、超大文件上传和长视频任务的人工体验测试。

测试环境可使用 `TASK_MOCK_MODE=true` 验证完整业务链路；生产环境必须为 `false`。
